# Индивидуальная лабораторная работа

_Архитектура Компьютера, 2 курс, 4 семестр_  
_Тема: Классификация стоматологических изображений (кариес / здоровый зуб) с помощью свёрточной нейронной сети_  
Год выполнения: **2026**  
**Текущее состояние:** _Проект готов. Сдан._

## Описание проекта

Цель работы – разработать и обучить нейросетевую модель, способную автоматически определять наличие кариеса на стоматологических изображениях.

**Основные задачи:**

- Подготовка датасета (разделение на обучающую и валидационную выборки, балансировка классов).
- Реализация пайплайна аугментации изображений для улучшения обобщающей способности.
- Использование предобученной модели ResNet18 (transfer learning) с заменой последнего полносвязного слоя на бинарный классификатор.
- Обучение с учётом дисбаланса классов (WeightedRandomSampler + веса в функции потерь).
- Оценка качества модели на валидационной выборке (accuracy, precision, recall, F1-score, confusion matrix).
- Создание скрипта для инференса на одном или нескольких изображениях.

**Результат:**  
Достигнута точность **93.88%** на валидации, F1-score для класса «кариес» – 0.96, для «здоровый» – 0.90. Модель сохранена в формате `.pth`, написаны утилиты для предсказаний на новых данных.

## Запуск

```bash
pip install -r requirements.txt

# Обучение
python src/train_model.py

# Предсказание для папки
python src/predict.py /путь/к/папке

# Предсказание для одного файла
python src/predict.py image.jpg
```

## Возможные флаги для запуска предсказания

```bash
# Указать не дефолтную модель
python src/predict.py --model /путь/к/модели

# Рекурсивно проверять подпапки
python src/predict.py --recursive

# Показать вероятность для каждого класса
python src/predict.py --show-probs

# Сохрнанить результаты в .csv (results.csv)
python src/predict.py --csv
```
